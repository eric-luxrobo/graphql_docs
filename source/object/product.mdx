import { Link } from 'gatsby'

### `Product`
> 상품 정보

| Name | Description |
| -------- | -------- | 
| id (String!)  | ID  | 
| title (String!)  | 제목  | 
| subTitle (String) | 서브제목 
| describe (String) | 설명
| mainImgs ([<Link to="/object/imageInfo/#ImageInfo">ImageInfo</Link>]) | 메인 이미지 리스트


### `ProductEdge`
> 상품 Edge

| Name | Description |
| -------- | -------- | 
| cursor (String!)  | 페이지 커서 ID  | 
| node (<Link to="/object/product/#Product">Product</Link>)  | 상품 정보  |


### `ProductConnection`
> 상품 리스트 정보

| Name | Description |
| -------- | -------- | 
| edges (String)  | 유저ID  | 
| nodes ([<Link to="/object/product/#Product">Product</Link>])  | 상품 정보 Array  |
| totalCount (Int) | 총 수 
| pageInfo (<Link to="/object/pageInfo/#PageInfo">PageInfo</Link>)  | 페이지 정보
