### `StatUserSummary`
> 통계 유저 Summary

| Name | Description |
| -------- | -------- | 
| userId (String!)  | userId  | 
| rate (<Link to="/object/stat/#StatRate">StatRate!</Link>) | 비율 정보
| studyTime (<Link to="/object/stat/#StatUserStudyTime">StatUserStudyTime!</Link>) | 공부시간 정보
| capacityMean (<Link to="/object/stat/#StatUserCapacityMean">StatUserCapacityMean!</Link>) | 평균 능려기
| capacityWeekly (<Link to="/object/stat/#StatUserCapacityWeekly">StatUserCapacityWeekly!</Link>) | 주간 능력치
| lessonCount (<Link to="/object/stat/#StatUserLessonCount">StatUserLessonCount!</Link>) | 강의 숫자 정보

### `StatUserLessonCount`
> 강의 숫자 정보 

| Name | Description |
| -------- | -------- | 
| lessonRegisterCount  (Int!)  | 수업 수강 숫자  | 
| lessonDoneCount  (Int!)  | 수강 완료 숫자  | 
| lessonOwnerCount: (Int!) | 내가 생성한 수업
| lessonOwnerStudentCount (Int!) | 내가 생성한 수업의 수강생
| lessonOwnerStudentMeanCount (Int!) | 내가 생성한 수업의 평균 수강생
