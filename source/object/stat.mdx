### `StatUserSummary`
> 통계 유저 Summary

| Name | Description |
| -------- | -------- | 
| userId (String!)  | userId  | 
| rate (StatRate!) | 비율 정보
| studyTime (StatUserStudyTime!) | 공부시간 정보
| capacityMean (StatUserCapacityMean!) | 평균 능려기
| capacityWeekly (StatUserCapacityWeekly!) | 주간 능력치


### `StatUserStudyTime`
> 공부시간 정보

| Name | Description |
| -------- | -------- | 
| coding (String!)  | 코딩영역  | 
| game (String!) | 게임
| english (String!) | 영어
| normal (String!) | 일반
| days (String!) | 학습일
