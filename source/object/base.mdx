### `UserActivity`
> 사용자 활동 정보

| Name | Description |
| -------- | -------- | 
| cLike (Int!)  | 강의 좋아요수  | 
| cBookmark (Int!)  | 걍의 북마크수  | 
| cView (Int!) | 강의 상세 페이지 뷰수
| cReview (Int!) | 강의 리뷰수 
| cLessonMember (Int!) | 수강 인원
| isLike (Boolean!) | 내가 좋아용 여부
| isBookmark (Boolean!) | 내가 북마크 여부
