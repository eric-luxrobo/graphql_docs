import { Link } from 'gatsby'

### `Lesson`
> 강의 정보

| Name | Description |
| -------- | -------- | 
| id (String!)  | 강의 ID  | 
| title (String!)  | 강의 제목  | 
| subTitle (String) | 강의 서브제목 
| describe (String) | 강의 설명
| videoURL (String) | 비디오 URL
| videoType (<Link to="/enum/lesson/#LessonVideoType">LessonVideoType</Link>) | 비디오 타입 (유튜브, 비메오..)
| classTime (Int) | 수업시간 (분기준)
| levelType (<Link to="/enum/lesson/#LessonLevelType">LevelType</Link>) | 강의 레벨
| groupType (<Link to="/enum/lesson/#LessonGroupType">GroupType</Link>) | 강의 그룹
| mainImgs ([<Link to="/object/imageInfo/#ImageInfo">ImageInfo</Link>]) | 강의 메인 이미지 리스트
| elementJSON (String) | 강의 속성정보 JSON
| userActivity (UserActivity!) | 사용자 활동 정보 좋아요..
| lessonPlans ([<Link to="/object/lesson/#LessonPlan">LessonPlan</Link>])  | 강의 계획 리스트
