## createUploadImage 
이미지 업로드 생성
<GraphiQL
  query="
  mutation createUploadImage( $input:CreateUploadImageInput! ){
    createUploadImage( input: $input ) {
      domain
      url
      width
      height
    }
  }
  "
  variables={{
    input: {
      id: "walter",
      serviceName:"lms",
      imageBase64:"data:image/jpeg;base64,/9j/4AAQSkZJRgABAQEASABIAAD"
    },
  }}
/>

Using Variables with Arguments
* id : 서비스 하위 키값
* serviceName : 서비스 이름
* imageBase64 : 이미지로드하여 base64로 인코딩한 이미지 string
