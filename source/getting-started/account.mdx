# members 
> 회원 리스트

**Parameter**

| key | 설명 | 타입 | 필수요청 |
| -------- | -------- | -------- | -------- | 
| page     | 페이즈값 디폴트 1   | int | true |
| size | 리스트 사이즈 개수 | int | true |
| field | 필드 요청 | int | true |
| word | 필터링 단어 | int | true |

<GraphiQL
  query="
    query members($page: Int!, $size: Int!) {
      members(page: $page, size: $size) {
        datas {
          id
          user_id
          user_name
          created_at
          updated_at
        }
      }
    }
  "
  variables={{
    page: 1,
    size: 10,
  }}
/>

**Response**
#### Members
> 회원리스트 

| key | 설명 | 타입 | 필수제공 |
| -------- | -------- | -------- | -------- |
| total_cnt     | 총수     | int | true |
| page_cnt | 페이지수 | int | true |
| datas | 회원 데이타 Array | [Member] | true | 


# member
> 회원 정보

**Parameter**

| key | 설명 | 타입 | 필수요청 |
| -------- | -------- | -------- | -------- | 
| userId     | 유저 ID   | string | true |


<GraphiQL
  query="
    query member($userId: String!) {
      member(userId: $userId) {
        id
        user_id
        user_name
        created_at
        updated_at
      }
    }
  "
  variables={{
    userId: "howto38",
  }}
/>

**Response**

#### Member
> 회원정보 

| key | 설명 | 타입 | 필수제공 |
| -------- | -------- | -------- | -------- |
| id     | member ID    | string | true |
| user_id | 사용자 ID | string | true |
| user_name | 사용자 이름 | string | true | 
| created_at | 생성 날짜 | string | true
| updated_at | 업데이트 날짜 | string | true
