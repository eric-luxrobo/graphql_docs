### `LessonPlanContentType`

```js:title=LessonPlanContentType
enum LessonPlanContentType {
  # 없음
  NONE_CONTENT
  # 기본 컨텐츠(이미지 또는 Html TEXT)
  BASE_CONTENT
  # 비디오 컨텐츠 
  VIDEO_CONTENT

}
```

### `LessonPlanContentViewType`
> 컨텐츠 화면 타입

```js:title=LessonPlanContentViewType
enum LessonPlanContentViewType {
  # 없음
  NONE_VIEW
  # 비디오 화면(앱이나 웹에서 화면을 비디오 화면으로 보여줌)
  VIDEO_VIEW
  # 코딩 화면(앱이나 웹에서 코딩화면을 보여준다.)
  CODING_VIEW
  # 기본형 전체화면
  NORMAL_VIEW
  # 가로줄 1 세로줄 아래 반
  HORIZON_FULL_VERTICAL_DOWN_HALF_VIEW
  # 세로줄 1 가로줄 오른쪽 반
  VERTICAL_FULL_HORIZON_RIGHT_HALF_VIEW
  # 세로줄 1
  VERTICAL_FULL_VIEW
  # 세로줄 2
  VERTICAL_TWO_FULL_VIEW
  # 가로줄 1 세로줄 1
  HORIZON_FULL_VERTICAL_FULL_VIEW
}
```
