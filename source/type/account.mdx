## `Members`
> 회원리스트 

| key | 설명 | 타입 | 필수제공 |
| -------- | -------- | -------- | -------- |
| total_cnt     | 총수     | int | true |
| page_cnt | 페이지수 | int | true |
| datas | 회원 데이타 Array | [<Link to="/type/account/#Member">Member</Link>] | true | 
```javascript
{
  "data": {
    "members": {
      "datas": [
        {
          "created_at": null,
          "id": "15",
          "updated_at": null,
          "user_id": "",
          "user_name": null
        }
      ]
    }
  }
}
```


## `Member`
> 회원정보 

| key | 설명 | 타입 | 필수제공 |
| -------- | -------- | -------- | -------- |
| id     | member ID    | string | true |
| user_id | 사용자 ID | string | true |
| user_name | 사용자 이름 | string | true | 
| created_at | 생성 날짜 | string | true
| updated_at | 업데이트 날짜 | string | true
