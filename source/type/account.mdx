### `AccountLoginInput`
> 로그인 입력정보

| Name | Description |
| -------- | -------- | 
| userId (String!)  | 유저ID  | 
| password (String!)  | 비밀번호  | 

### `AccountCode` 
> 계정 코드 정보

| Name | Description |
| -------- | -------- | 
| code (Int) | 코드 정보
| name (String) | 코드 이름
| description (String) | 코드 설명

| Name | Description |
