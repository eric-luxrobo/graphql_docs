> 계정 리스트 

| Name | Description |
| -------- | -------- | 
| totalCnt (Int!)  | 총수  | 
| pageCnt (Int!)  | 페이지수  | 
| nodes ([<Link to="/type/account/#AccountMember">AccountMember</Link>])  | 회원 데이타 Array  | 

```javascript
{
  "data": {
    "members": {
      "nodes": [
        {
          "createdAt": null,
          "id": "15",

        }
      ]

  }
}
```

## `AccountMember`
> 회원정보 

| Name | Description |
| -------- | -------- | 
| id (String!)  | member ID  | 
| userId (String!)  | 사용자 ID  | 
| userName (String!)  | 사용자 이름  | 
| createdAt (String!)  | 생성 날짜 | 


## `AuthCheckAccountUserEmailInput`
> 이메일 인증 체크 입력정보

| Name | Description |
| -------- | -------- | 
| id (ID!)  | ID  | 
| authCode (String!)  | 인증코드  |

| userEmail (String!) | 이메일 |
