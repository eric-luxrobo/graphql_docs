## `CreateUploadImageInput`
> 이미지 업로드 입력 파라미터 

| Name | Description |
| -------- | -------- | 
| id (String!)  | 서비스 하위 키값   | 
| serviceName (String!)  | 서비스 이름  | 
| imageBase64 (String!)  | 이미지로드하여 base64로 인코딩한 이미지  | 


## `ImageInfo`
> 이미지 정보

| Name | Description |
| -------- | -------- | 
| domain (Int!)  | 이미지 도메인    | 
| url (Int!)  | 이미지 Full URL | 
| width (Int!)  | 이미지 넓이  | 
| height (Int!)  | 이미지 높이  |
