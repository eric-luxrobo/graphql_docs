### `CreateLessonInput`
> 강의 등록 

| Name | Description |
| -------- | -------- | 
| name (String!)  | 강의 제목  | 
| subTitle (String) | 강의 서브제목 
| describe (String) | 강의 설명
| classTime (Int) | 수업시간 (분기준)
| levelType (LevelType) | 강의 레벨
| groupType (GroupType) | 강의 그룹
| mainImgs ([ImageInfoInput]) | 강의 메인 이미지 리스트
| elementJSON (String) | 강의 속성정보 JSON


### `Lesson`
> 강의 정보

| Name | Description |
| -------- | -------- | 
| id (String!)  | 강의 ID  |

> 강의 리스트 정보

| Name | Description |
| -------- | -------- |

| nodes (String!)  | 비밀번호  |
