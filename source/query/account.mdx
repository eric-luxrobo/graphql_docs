# `members` 
Type : <Link to="/type/account/#Members/">Members</Link>

> Role: [관리자] 회원정보 리스트를 보여줍니다. 

**Arguments**

| Name | Description |
| -------- | -------- | 
| page (Int)  | 페이즈값 디폴트 1  | 
| size (Int) | 리스트 사이즈 개수 | 
| field (Int) | 필드 요청 |
| word (Int) | 필터링 단어 | 

<GraphiQL
  query="
    query members($page: Int!, $size: Int!) {
      members(page: $page, size: $size) {
        total_cnt
        page_cnt
        datas {
          id
          user_id
          user_name
          created_at
          updated_at
        }
      }
    }
  "
  variables={{
    page: 1,
    size: 10,
  }}
/>
