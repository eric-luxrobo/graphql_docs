import { Link } from 'gatsby'

### `AuthAccountUserEmailInput`
> 이메일 인증 발송요청.

| Name | Description |
| -------- | -------- | 
| userEmail (String!) | 발송 이메일


### `AuthCheckAccountUserEmailInput`
> 이메일 인증 체크 입력정보

| Name | Description |
| -------- | -------- | 
| id (ID!)  |  이메일 인증발송 요청후 리턴된 고유ID | 
| authCode (String!)  | 메일에서 확인된 인증코드 값  | 



### `CreateAccountMemberInput`
> 회원가입 입력 정보

| Name | Description |
| -------- | -------- | 
| userId (String!)  | ID  | 
| password (String!)  | 비밀번호  | 
| userName (String!) | 이름 |
| userEmail (String!) | 이메일 | 
| gender (<Link to="/enum/account/#AccountGenderEnumType">AccountGenderEnumType</Link>)  | 계정  성별 |
| accountCode (Int!) | 계정 타입
| memberCode (Int!) | 멤버 타입
| provider (<Link to="/enum/account/#AccountProviderEnumType">AccountProviderEnumType</Link>)  | 계정 로그인 서비스 타입

### `LoginAccountInput`
> 로그인 입력정보

| Name | Description |
| -------- | -------- | 
| userId (String!)  | 유저ID  | 
| password (String!)  | 비밀번호  | 


### `ResetAccountPasswordInput`
> 회원 비밀번호 Reset 입력 정보

| Name | Description |
| -------- | -------- | 
| userEmail (String!) | 이메일


### `OAuthLoginAccountInput`
> 소셜 로그인 입력 정보

| Name | Description |
| -------- | -------- | 
| userId (String!)  | 소셜 ID  | 
| userName (String!)  | 소셜 이름  | 
| userEmail (String!) | 소셜 이메일 
| provider (<Link to="/enum/account/#AccountProviderEnumType">AccountProviderEnumType</Link>)  | 계정 로그인 서비스 타입
| accountCode (Int!) | 계정 타입
| memberCode (Int!) | 멤버 타입
| accessToken (String!) | 소셜 로그인 AccessToken 정보
| refreshToken (String) | 소셜 로그인 refreshToken 연장 토큰 정보
