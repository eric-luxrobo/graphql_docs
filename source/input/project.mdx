### `CreateProjectInput`
> 프로젝트 입력 정보

| Name | Description |
| -------- | -------- | 
| title (String!)  | 프로젝트 제목  | 
| userKey (String!) | 유저 UUID
| codeType (ProjectCodeType!) | 프로젝트 타입 (SCRATCH,ENTRY)
| jsonData (String) | 프로젝트 코딩 데이타 정보
| thumb (ImageInfoInput) | 이미지 정보
