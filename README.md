
# Luxrobo API Docs 
luxrobo 개발 API문서정리 사이트입니다.
software팀에서는 graphql를 API로 사용하고 있으며 서비스마다 grpc를 통해서 통신하고 있습니다.
문서는 graphql type에 대한정의와 grpc문서를 위주로 작성하였습니다.

또한 서비스와의 관계도도 추가예정
